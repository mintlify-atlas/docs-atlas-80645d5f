## Welcome to Click Creators Scraper Client

Click Creators Scraper Client is a comprehensive Instagram marketing automation platform designed to streamline follower acquisition, campaign management, and virtual assistant (VA) coordination workflows. Built with Next.js 14 and TypeScript, it provides a modern, scalable solution for managing Instagram marketing campaigns at scale.

## What is Click Creators Scraper Client?

This platform automates the entire workflow of Instagram follower scraping, from initial data collection to distribution among virtual assistants:

1. **Scrape Instagram followers** from source accounts using Apify integration
2. **Filter profiles by gender** using name-based detection algorithms
3. **Store and deduplicate** profiles in a Supabase PostgreSQL database
4. **Create campaigns** with daily batches of unused profiles (14,400 per campaign)
5. **Distribute to VAs** across 80 virtual assistant tables
6. **Sync to Airtable** for easy VA access and workflow management
7. **Manage lifecycle** with automated 7-day campaign cleanup

## Architecture Overview

The platform follows a modern full-stack architecture:

```
Source Profiles (Instagram IDs)
        ↓
   Apify Scraper (Extract Followers)
        ↓
   Gender Detection (Filter Males)
        ↓
   Supabase Storage (Deduplication)
        ↓
   Daily Selection (14,400 profiles)
        ↓
   VA Distribution (80 tables × 180)
        ↓
   Airtable Sync (VA Access)
```

Each component is designed for scalability, reliability, and multi-tenant isolation, ensuring secure and efficient operations across multiple concurrent campaigns.
