## Overview

The `daily_assignments` table stores the distribution of profiles to **80 Virtual Assistants (VAs)**. Each campaign assigns **14,400 profiles** with **180 profiles per VA**.

<Info>
This table serves as the bridge between campaigns and Airtable VA tables, tracking which profiles are assigned to which VAs.
</Info>

## Schema Definition

```sql
CREATE TABLE daily_assignments (
  assignment_id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  campaign_id UUID REFERENCES campaigns(campaign_id),
  id TEXT NOT NULL,  -- Instagram profile ID
  username TEXT NOT NULL,
  full_name TEXT,
  va_table_number INTEGER NOT NULL,
  position INTEGER NOT NULL,
  status TEXT DEFAULT 'pending',
  job_id UUID REFERENCES scraping_jobs(job_id),
  base_id TEXT NOT NULL,  -- Multi-tenant isolation key
  assigned_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMPTZ
);
```

## Distribution Algorithm

### Equal Distribution (180 per VA)

```typescript
function distributeProfiles(usernames: GlobalUsername[], numVAs = 80) {
  const PROFILES_PER_VA = 180
  const assignments: DailyAssignment[] = []
  
  for (let vaNum = 1; vaNum <= numVAs; vaNum++) {
    const startIdx = (vaNum - 1) * PROFILES_PER_VA
    const vaProfiles = usernames.slice(startIdx, startIdx + PROFILES_PER_VA)
    
    for (let i = 0; i < vaProfiles.length; i++) {
      assignments.push({
        campaign_id: campaignId,
        id: vaProfiles[i].id,
        username: vaProfiles[i].username,
        full_name: vaProfiles[i].full_name,
        va_table_number: vaNum,
        position: i + 1,
        status: 'pending',
        job_id: jobId,
        base_id: baseId,
        assigned_at: new Date().toISOString()
      })
    }
  }
  
  return assignments
}
```

## Airtable Sync

### Sync VA Table to Airtable

```typescript
async function syncVATableToAirtable(
  vaNumber: number,
  assignments: DailyAssignment[],
  airtableBaseId: string
) {
  const airtable = new Airtable({ apiKey: process.env.AIRTABLE_API_KEY })
  const base = airtable.base(airtableBaseId)
  const tableName = `Table ${vaNumber}`
  
  // Prepare records for Airtable
  const records = assignments.map(a => ({
    fields: {
      'Username': a.username,
      'Full Name': a.full_name || '',
      'Position': a.position,
      'Status': a.status,
      'Profile ID': a.id
    }
  }))
  
  // Batch create (Airtable limit: 10 records per request)
  const BATCH_SIZE = 10
  for (let i = 0; i < records.length; i += BATCH_SIZE) {
    const batch = records.slice(i, i + BATCH_SIZE)
    await base(tableName).create(batch)
    
    // Rate limiting
    await new Promise(resolve => setTimeout(resolve, 200))
  }
}
```

### Sync All VA Tables

```typescript
async function syncCampaignToAirtable(campaignId: string, baseId: string) {
  const { data: assignments } = await supabase
    .from('daily_assignments')
    .select('*')
    .eq('campaign_id', campaignId)
    .eq('base_id', baseId)
    .order('va_table_number')
  
  // Group by VA table
  const grouped = assignments.reduce((acc, a) => {
    if (!acc[a.va_table_number]) acc[a.va_table_number] = []
    acc[a.va_table_number].push(a)
    return acc
  }, {})
  
  // Sync each VA table
  for (const [vaNum, vaAssignments] of Object.entries(grouped)) {
    console.log(`Syncing VA ${vaNum}...`)
    await syncVATableToAirtable(Number(vaNum), vaAssignments, baseId)
  }
}
```

## Lifecycle Management

### Campaign Cleanup (7 Days)

```typescript
async function cleanupCampaignAssignments(
  campaignId: string,
  baseId: string
) {
  // Get all assignments for campaign
  const { data: assignments } = await supabase
    .from('daily_assignments')
    .select('id')
    .eq('campaign_id', campaignId)
    .eq('base_id', baseId)
  
  const profileIds = assignments.map(a => a.id)
  
  // Reset 'used' flag in global_usernames
  await supabase
    .from('global_usernames')
    .update({ used: false, used_at: null })
    .in('id', profileIds)
    .eq('base_id', baseId)
  
  // Delete assignments (cascade from campaign delete)
  await supabase
    .from('daily_assignments')
    .delete()
    .eq('campaign_id', campaignId)
    .eq('base_id', baseId)
  
  console.log(`Freed ${profileIds.length} profiles for reuse`)
}
```

## Best Practices

<Check>**Batch insert assignments** - Insert 1000 records at a time for better performance</Check>

<Check>**Validate VA distribution** - Ensure each VA gets exactly 180 profiles</Check>

<Check>**Use CASCADE delete** - Let foreign key cascade handle cleanup when campaigns are deleted</Check>

<Warning>**Always include base_id** - Critical for multi-tenant isolation in all queries</Warning>

<Warning>**Handle Airtable rate limits** - Add delays between batch uploads (200ms recommended)</Warning>
