## Overview

The `source_profiles` table stores Instagram account usernames that serve as **sources** for follower scraping. These are typically influencer accounts or target accounts whose followers will be extracted and analyzed.

<Info>
Source profiles are managed through the "Edit Source Profiles" dialog in the Dependencies Card component.
</Info>

## Schema Definition

```sql
CREATE TABLE source_profiles (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  username TEXT NOT NULL UNIQUE,
  base_id TEXT NOT NULL,  -- Multi-tenant isolation key
  created_at TIMESTAMP DEFAULT NOW()
);
```

### Get All Source Profiles

```typescript
import { supabase } from '@/lib/supabase'
```

## Username Validation

### Extract Username from URL

```typescript
function extractUsername(input: string): string {
  // Handle Instagram URLs
  const urlPattern = /instagram\.com\/([a-zA-Z0-9._]+)/
  const match = input.match(urlPattern)
  
  if (match) {
    return match[1] // Extract username from URL
  }
  
  // Remove @ symbol if present
  return input.replace('@', '').trim()
}

// Usage
const username = extractUsername('https://instagram.com/taylorswift')
// Returns: "taylorswift"
```

### Validate Username Format

```typescript
function isValidInstagramUsername(username: string): boolean {
  // Instagram usernames: 1-30 chars, alphanumeric + . and _
  const pattern = /^[a-zA-Z0-9._]{1,30}$/
  return pattern.test(username)
}
```

## UI Integration

### Dependencies Card Component

Source profiles are managed through the **Edit Source Profiles** dialog:

```typescript
// components/edit-source-profiles-dialog.tsx
const handleAddProfile = async () => {
  const username = extractUsername(inputValue)
  
  if (!isValidInstagramUsername(username)) {
    toast.error('Invalid Instagram username')
    return
  }
  
  const { error } = await client
    .from('source_profiles')
    .insert({ username, base_id: baseId })
  
  if (error) {
    if (error.code === '23505') { // Unique violation
      toast.error('Username already exists')
    } else {
      toast.error('Failed to add profile')
    }
  } else {
    toast.success('Profile added')
  }
}
```

## Scraping Workflow

### 1. Add Source Profiles

Users add Instagram accounts to scrape:

```typescript
// Add multiple source profiles
const usernames = ['taylorswift', 'arianagrande', 'selenagomez']

for (const username of usernames) {
  await client
    .from('source_profiles')
    .insert({ username, base_id: baseId })
}
```

### 2. Fetch for Scraping

When scraping starts, fetch all source profiles:

```typescript
const { data: profiles } = await client
  .from('source_profiles')
  .select('username')
  .eq('base_id', baseId)

const usernames = profiles.map(p => p.username)
// Pass to scraping API
```

### 3. API Integration

Send source accounts to the backend scraper:

```typescript
const response = await fetch('/api/scrape-followers', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    accounts: usernames, // From source_profiles
    targetGender: 'male'
  })
})
```

## Error Handling

### Duplicate Username

```typescript
const { error } = await client
  .from('source_profiles')
  .insert({ username, base_id: baseId })

if (error?.code === '23505') {
  console.error('Username already exists in source profiles')
  toast.error('This Instagram account is already in your source list')
}
```

### Invalid Username

```typescript
if (!isValidInstagramUsername(username)) {
  toast.error('Invalid Instagram username. Use only letters, numbers, dots, and underscores.')
  return
}
```

### Missing base_id

```typescript
if (!baseId) {
  console.error('No active job found. Cannot add source profile.')
  toast.error('Please create a scraping job first')
  return
}
```
